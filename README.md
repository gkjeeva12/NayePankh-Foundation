# NayePankh Foundation Website

A responsive multi-page NGO website developed using HTML, CSS, and JavaScript. The website is designed to represent the NayePankh Foundation and showcase its mission, programs, volunteer opportunities, and contact information.

---

## 📌 Project Overview

NayePankh Foundation is a non-profit organization dedicated to empowering underprivileged children and communities through education, healthcare, and social development initiatives.

This website provides information about the foundation and encourages visitors to participate as volunteers and supporters.

---

## 🚀 Features

- Responsive Design
- Multi-Page Website
- Modern User Interface
- Home Page with Hero Section
- About Us Page
- Programs Page
- Volunteer Registration Form
- Contact Form
